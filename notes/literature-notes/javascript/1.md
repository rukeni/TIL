# 자바스크립트 어휘 구조

- 대소문자 구분, 스페이스, 줄바꿈
- 주석
- 리터럴
- 식별자와 예약어
- 유니코드
- 선택 사항인 세미콜론

## 주석

```js

// 한줄 주석
/* 주석 */
/* 
 * 여러줄 주석 각 행의 * 문자는 꼭 써야하는 것은 아님
 */

```

## 리터럴
프로그램 안에 직접 쓴 데이터 값

## 식별자와 예약어
식별자는 반드시 알파벳 사용을 권한다. 밑줄과 달러 기호포함

- 상수
- 변수
- 프로퍼티
- 함수
- 클래스

예약어로 지정된 키워드는 이름으로 사용할 수 없음
(객체의 프로퍼티로는 쓸 수 있음)

## 유니코드
유니코드 문자는 모두 문자열이나 주석에 사용할 수 있음

## 세미콜론
다른 언어와 마찬가지로 실행의 구분이 된다.
